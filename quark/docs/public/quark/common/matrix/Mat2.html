    Struct Mat2

    Generic 2x2 matrix struct supporting both integer and floating-point types.

    struct Mat2(T)
      
    if (__traits(isIntegral, T) || __traits(isFloating, T));

    Supports

    - Elements m00, m01, m10, and m11 - Array of 2 arrays of 2 elements each where the first array is the first row
