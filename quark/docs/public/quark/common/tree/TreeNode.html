    Class TreeNode

    Generic tree node class storing data and child nodes.

    class TreeNode(T);

    Constructors

    NameDescription
    this (data, children) Creates a tree node with data and initial children.

    Fields

    NameTypeDescription
    children TreeNode!T[]Array of child tree nodes
    data TThe data stored in this tree node

    Methods

    NameDescription
    _traverseNode (action)
    addChild (child) Adds a child node to this node.
    depth () Gets the depth/level of this node in the tree.
    findAllOf (predicate) Finds all tree nodes whose data satisfy the given predicate.
    findFirstOf (predicate) Finds the first tree node whose data satisfies the given predicate.
    isLeaf () Checks if this node is a leaf (has no children).
    opIndex (index) Array-style access to child nodes.
    pathTo (target) Finds the path from this node to a target node.
    removeChild (child) Removes a child node from this node.
    size () Gets the total number of nodes in the subtree rooted at this node.
    traverse (action) Traverses the tree applying an action to each node's data.

    Example

    Test complex tree operations. 

    auto root = new TreeNode!int(10);

foreach (i; 1 .. 4)
{
    auto child = new TreeNode!int(i);
    root.addChild(child);

    foreach (j; 1 .. 3)
    {
        auto grandchild = new TreeNode!int(i * 10 + j);
        child.addChild(grandchild);
    }
}

assert(root.size() == 10, "Complex tree should have 10 nodes");
assert(root.depth() == 3, "Complex tree should have depth 3");

auto evenNodes = root.findAllOf((int data) => data % 2 == 0);
assert(evenNodes.length == 5, "Should find 5 even-valued nodes");
