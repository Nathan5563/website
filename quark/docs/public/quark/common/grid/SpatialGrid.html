    Class SpatialGrid

    A simple 2D Spatial Hash Grid for broadphase collision detection.

    class SpatialGrid(T);

    Maps world coordinates to grid cells and stores a list of unique items (referenced by ID) in each cell.

    Constructors

    NameDescription
    this (cellSize) Creates a new SpatialGrid.

    Fields

    Properties

    NameTypeDescription
    cellCount[get] size_tGets the current number of cells occupied in the grid.
    cellSize[get] floatGets the cell size.

    Methods

    NameDescription
    clear () Clears the entire grid, removing all stored items.
    getCellKey (position) Gets the grid key (cell coordinates) for a given world position.
    insert (itemId, minWorld, maxWorld) Inserts an item (by ID) into the grid based on its world-space AABB. The item is added to all cells that the AABB overlaps.
    query (minWorld, maxWorld) Queries the grid for all items in cells overlapping a given AABB.
    query (center, radius) Queries the grid for all items in cells overlapping a given circle. The circle's bounding box (AABB) is used to determine the cells to check.
