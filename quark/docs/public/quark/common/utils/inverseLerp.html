    Function inverseLerp

    Linear interpolation factor between two values.

    float inverseLerp(
      float a,
      float b,
      float value
    );

    Parameters

    NameDescription
    a Start value
    b End value
    value The value to find the factor for, must be in [a, b]

    Returns

    The interpolation factor in [0.0, 1.0]
