    Class PhysicsWorld

    Main physics world that coordinates collision detection and physics simulation.

    class PhysicsWorld;

    The PhysicsWorld manages all physics entities, performs collision detection using broadphase and narrowphase algorithms, and handles physics integration. It maintains the connection between ECS entities and physics simulation.
