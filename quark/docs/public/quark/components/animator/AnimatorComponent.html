    Class AnimatorComponent

    Component for managing sprite-based animations.

    class AnimatorComponent
      : Component;

    The AnimatorComponent controls frame-based animation by managing a sequence of frames from a sprite sheet. It updates the SpriteRendererComponent's source rectangle to display the correct frame based on timing and animation state.
