    Class SpriteRendererComponent

    Component for rendering a sprite texture on an entity.

    class SpriteRendererComponent
      : Component;

    The SpriteRendererComponent displays a 2D texture at the entity's transform position. It supports advanced rendering features including rotation, flipping, alpha blending, and sprite sheet source rectangle selection. The texture is loaded asynchronously via the resource manager.
