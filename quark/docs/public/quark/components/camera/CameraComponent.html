    Class CameraComponent

    Component for controlling the viewport and camera transformations in 2D space.

    class CameraComponent
      : Component;

    The camera uses the entity's TransformComponent for position and rotation. Multiple cameras can exist but typically only one is active at a time. The camera entity should have a TransformComponent attached.
