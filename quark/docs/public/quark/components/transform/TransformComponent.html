    Class TransformComponent

    Component that defines an entity's position, rotation, and scale in 2D space.

    class TransformComponent
      : Component;

    The Transform component is the fundamental building block for positioning entities in the scene, providing position, rotation, and scaling, along with associated methods.

    Constructors

    NameDescription
    this (position, rotation, scale, size) Creates a new Transform component (defaults to position(0,0), rotation(0), scale(1,1), size(32,32)).

    Example

    Test basic transform creation and default values. 

    import quark.common.utils : fequals;

auto transform = new TransformComponent();
assert(transform.position.x.fequals(0.0f) && transform.position.y.fequals(0.0f),
    "Default position should be (0, 0)");
assert(transform.rotation.fequals(0.0f), "Default rotation should be 0");
assert(transform.scale.x.fequals(1.0f) && transform.scale.y.fequals(1.0f),
    "Default scale should be (1, 1)");
assert(transform.size.x.fequals(32.0f) && transform.size.y.fequals(32.0f),
    "Default size should be (32, 32)");
