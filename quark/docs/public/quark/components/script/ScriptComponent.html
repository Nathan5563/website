    Class ScriptComponent

    Component that executes QScript files for entity behavior.

    class ScriptComponent
      : Component;

    The ScriptComponent loads and executes QScript files using the interpreter in the SceneManager. Scripts can define collision callbacks and other entity behaviors that are called during gameplay.
