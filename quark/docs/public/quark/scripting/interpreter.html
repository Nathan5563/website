    Module quark.scripting.interpreter

    @file interpreter.d

    Interpreter for QScript scripting language.

    This module provides runtime execution for QScript (.qscr) files, evaluating the Abstract Syntax Tree produced by the parser. The interpreter manages variable scopes, function calls, control flow, and provides built-in functions that expose the Quark engine's public APIs to scripts.
