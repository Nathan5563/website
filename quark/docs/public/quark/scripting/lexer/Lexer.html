    Class Lexer

    Lexical analyzer that converts source code into tokens.

    class Lexer;

    The lexer reads through QScript source code character by character, identifying and producing tokens according to the QScript grammar. It handles comments, whitespace, and provides error reporting with line and column information.

    Test basic tokenization of simple tokens. 

    auto lexer = new Lexer("( ) { } ; ,");
assert(lexer.tokenize(), "Should tokenize successfully");
assert(lexer.tokens.length == 7, "Should have 6 tokens plus EOF");
assert(lexer.tokens[0].type == TokenType.LPAREN, "First token should be LPAREN");
assert(lexer.tokens[1].type == TokenType.RPAREN, "Second token should be RPAREN");
