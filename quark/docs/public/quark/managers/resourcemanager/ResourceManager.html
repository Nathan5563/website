    Class ResourceManager

    Thread-safe singleton manager for game resource loading and caching.

    class ResourceManager
      : Manager;

    The ResourceManager handles loading textures and sounds from disk, caching them for efficient reuse. All public methods are thread-safe through mutex synchronization.
