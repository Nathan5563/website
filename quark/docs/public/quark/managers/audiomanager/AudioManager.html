    Class AudioManager

    Thread-safe singleton manager for audio playback and streaming.

    class AudioManager
      : Manager;

    The AudioManager handles loading, playing, and looping audio files using SDL3 audio streams. All public methods are thread-safe through mutex synchronization. Each sound is preloaded with its own audio stream for efficient playback.
