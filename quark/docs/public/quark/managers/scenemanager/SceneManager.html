    Class SceneManager

    Thread-safe singleton manager for game scene management and transitions.

    class SceneManager
      : Manager;

    The SceneManager maintains a collection of scenes indexed by their unique IDs and tracks the currently active scene. It provides methods for scene switching with proper cleanup, and forwards all lifecycle methods (preUpdate, postUpdate, preRender, postRender) to the active scene. Designed to be called from the main game loop while remaining thread-safe for access from other systems.
