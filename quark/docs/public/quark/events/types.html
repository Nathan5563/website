    Classes

    NameDescription
    E_CollisionEnter Collision event sent when two entities first collide.
    E_CollisionExit Collision event sent when two entities stop colliding.
    E_CollisionStay Collision event sent when two entities continue to collide from previous frame. Only fired when there are significant changes in collision properties.
    E_LoadResources Request to load resources from a JSON config file.
    E_LoadSound Request to load a sound from disk.
    E_LoadTexture Request to load a texture from disk.
    E_LoopSound Request to start looping a sound.
    E_PlaySound Request to play a sound once.
    E_RegisterBody Request to register an entity with the physics system.
    E_StopAllLoopingSounds Request to stop all looping sounds.
    E_StopLoopingSound Request to stop a looping sound.
    E_TriggerEnter Trigger event sent when a trigger collider overlaps with another collider.
    E_UnregisterBody Request to unregister an entity from the physics system.
