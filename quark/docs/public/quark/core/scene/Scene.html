    Class Scene

    Represents a game scene containing a hierarchy of entities.

    class Scene;

    The Scene class manages a tree of entities and their lifecycle. It provides methods to add, remove, and find entities, as well as to trigger update and render cycles across all entities in the scene via depth-first traversal. Scenes are typically managed by a SceneManager.
